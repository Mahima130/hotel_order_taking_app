# Restaurant & Hotel Order Management System

A Flutter-based mobile application designed to streamline restaurant and hotel operations by enabling waiters to manage tables, take orders, and track revenue in real time. The system provides an intuitive UI for table assignment, order management, and performance tracking.

## Tech Stack
- Flutter (Dart)
- Firebase (Firestore / Realtime Database)
- REST-style data handling

## Key Features
- Smart table management with real-time table availability
- QR-based or manual table assignment for waiters
- Order taking and live order updates for each table
- Individual order tracking per table
- Status management for orders (Pending, Completed, Cancelled)
- Automatic calculation of revenue generated per table
- Centralized dashboard for active and completed orders
- Clean, responsive, and user-friendly UI optimized for daily restaurant operations

## Table & Order Management
- Assign empty tables to customers instantly
- Maintain separate order records for each table
- Track total bill amount generated by every table
- Store order history including completed and cancelled orders
- Real-time synchronization across devices using Firebase

## Use Case
Designed for restaurants and hotels to help waiters and managers efficiently handle table assignments, order flow, and revenue tracking while reducing manual errors and improving service speed.

## How to Run
1. Clone the repository
2. Run `flutter pub get`
3. Configure Firebase project and add credentials
4. Run `flutter run`
